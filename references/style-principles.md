# Style Principles

The default profile is a warm editorial single-file HTML report.

## Layout

- Single main column.
- `.page` max width: `1180px`.
- Body padding: `56px 32px 120px`.
- Aligned edges across header, context box, TL;DR, summary, sections, diagrams, and tables.

## Typography

- Body: `15px`.
- H1: `38px`.
- H2: `26px`.
- Headings: serif.
- Body: system sans.
- Monospace: code, commands, hashes, field names, and technical IDs only.

## Structure

- Answer first.
- Summary cards second.
- Evidence follows the decision.
- Raw material is folded.
- Verification and limitations are visible near the end.

## Data honesty

- Do not invent figures, file paths, commands, or observations.
- If an input is missing, say so.
- If a check was not run, record it as a limitation.
