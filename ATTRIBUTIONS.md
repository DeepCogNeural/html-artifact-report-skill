# Attributions

This repository is a clean-room implementation. It absorbs ideas and patterns from related open-source work, but v1 does not copy source code, templates, screenshots, or assets from those projects.

## Inspired by

- `haidang1810/md2html` — MIT license, https://github.com/haidang1810/md2html. Inspired semantic Markdown-to-HTML agent workflow, component catalog discipline, and single-file HTML delivery.
- `alchaincyf/huashu-md-html` — MIT license, https://github.com/alchaincyf/huashu-md-html. Inspired the principle that Markdown is source and HTML is an output artifact, plus explicit anti-slop design rules.
- `nexu-io/html-anything` — Apache-2.0 license, https://github.com/nexu-io/html-anything. Inspired skill metadata, golden examples, and local agent artifact thinking.
- Claude Code HTML artifact workflow, https://claude.com/blog/using-claude-code-the-unreasonable-effectiveness-of-html. Inspired the high-level preference for rich, readable, shareable HTML artifacts for complex agent work.
